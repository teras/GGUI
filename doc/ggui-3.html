3. Installation instructions

Since GGUI version 0.5.0 there is no need to install the program at all. All required datafiles and pictures are already inside the JAR file. On run time the program copies the required files into the directory ~/.ggui automatically. The option to copy the files somewhere system wide is not yet implemented.

If you still have problems/questions, you are welcomed to ask me.

The GGUI application includes the Wizard source code (by me). This source code is no longer given alone but it is included inside the GGUI package.

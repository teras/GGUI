9. Other Projects

I have another project in mind which will shrink even more the usage of the command-prompt: to create a similar programming interface for manipulating the output of any program. This interface will track down the output results and act as desired. It is a very difficult problem and you are ALL welcomed for suggestions and talking.

Last but not least is to change the actual ggui data file format. Me and Maik Roeder are thinking of a new prototype, maybe something in XML... The future will show!

6. The Creator application

Here you will find information how to use the Creator application to create new datafiles or edit old ones.

6.1 Screen display

The screen is divided into many sections.

Left - Top

A list box with all the GGUI pages and objects inside. Use it as a map to navigate through the various objects.

Left - Bottom

The tools you can use to create a data file. First, you have to select a tool by clicking on it. Then you have to add it. Every option of the tool has a small text describing what this option does. I hope you'll not have any problems with these - after all experiment a little bit!

  • Page. Every wizard is divided into pages. Use this tool to add one more page after the selected item.
  • Info. Use this object if you want to display some info text on the wizard at this position.
  • Boolean. Use this object if you want to define a boolean (e.g. on/off or yes/no) parameter/switch.
  • Chooser. Use this object if you want to define a multiple choice parameter. Sometimes this is preferred than Boolean, because it can define a 3-state "yes/no/default" value.
  • FileChooser. Use this object if the wizard needs to collect a filename parameter.
  • FreeText. Use this object if the parameter needs a textual input.
  • 3DLine. This is not a parameter object. It just creates a fancy 3D line at the selected position.

Bottom - Left

Here you can see the name of the selected object.

Top

  • Add button: Add an object (taken from the tolls below) after the selected object.
  • Remove button: Remove the selected object.

Bottom - Right

  • Cancel button: Exit Creator and don't save anything.
  • Remove button: Exit Creator and save current file.

Central

Here you can find the parameters for any selected object.
